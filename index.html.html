#01 for 반복문 + continue — 특정 인덱스만 출력
forcontinue
▾ 
# 01
star = ['A', 'B', 'C', 'D', 'E']
for i in range(5):
    if i != 3:
        continue
    print(star[i])
출력 D

단계별 실행 흐름

1
star는 5개짜리 리스트. 인덱스는 0~4, 값은 A~E.
2
range(5)는 0,1,2,3,4 순서로 i에 넣음.
3
i != 3이면 continue → 즉시 다음 반복으로 건너뜀. print는 실행 안 됨.
4
i == 3일 때만 조건이 False이므로 continue를 건너뛰고 print(star[3]) 실행 → D
💡 continue 란?
반복문 안에서 continue를 만나면 아래 코드를 건너뛰고 즉시 다음 반복으로 이동합니다.
break가 반복문을 완전히 종료하는 것과 다릅니다.
#02 print(*) 언패킹 + 제너레이터 표현식
언패킹제너레이터
▾ 
# 02
star = ['① A', '② B', '③ C', '④ D', '⑤ E']
print(*(x for x in star if x[0] not in '①②④⑤'))
출력 ③ C

단계별 실행 흐름

1
각 원소의 x[0]는 첫 글자를 의미. 예: '③ C'[0] → '③'
2
필터 조건: x[0] not in '①②④⑤' — 첫 글자가 저 4개에 없으면 통과.
3
'③'는 '①②④⑤'에 없으므로 '③ C'만 통과 → 제너레이터는 '③ C' 하나만 생성.
4
print(*...)의 *가 제너레이터를 풀어서 출력 → ③ C
⚡ 언패킹(Unpacking) — print(*iterable)
*는 리스트나 이터러블을 낱개로 풀어주는 연산자입니다.

a = [1, 2, 3] print(a) # → [1, 2, 3] (리스트 그대로) print(*a) # → 1 2 3 (개별 인자로 풀려서 전달)
즉 print(*a)는 print(1, 2, 3)과 완전히 동일합니다.
💡 제너레이터 표현식 vs 리스트 컴프리헨션
[x for x in ...] → 대괄호: 리스트 즉시 생성
(x for x in ...) → 괄호: 제너레이터, 필요할 때만 값 생산 (메모리 효율적)

▾

두 코드는 결과가 동일하게 D를 출력합니다. 방식이 다를 뿐:

• #01 (for + continue): 모든 i(0~4)를 순회하되, i≠3이면 건너뜀
• #03 (while + break): i==3을 찾으면 출력 후 즉시 탈출
#04 소수(prime) 개수 세기 — 한줄 심화 코드
한줄코딩all()수학
▾ 
# 04
sum(all(i%j for j in range(2,int(i**.5)+1))for i in range(3,10**6,2))+1
출력 78498 (100만 이하 소수의 개수)

분해해서 이해하기

이 코드를 풀어 쓰면 아래와 같습니다:

count = 1 # 2 (짝수 소수) 미리 포함 for i in range(3, 10**6, 2): # 홀수만 (3,5,7,9...) is_prime = True for j in range(2, int(i**0.5)+1): if i % j == 0: # 나누어지면 소수 아님 is_prime = False break if is_prime: count += 1
⚡ 핵심 테크닉 3가지
① i**0.5 = √i 로 검사 범위 줄이기
어떤 수 i가 소수가 아니라면 √i 이하의 약수가 반드시 존재합니다. 그래서 2~√i 범위만 검사해도 충분합니다.

② i%j 가 bool처럼 사용되는 원리
i%j가 0이면 False (j로 나누어짐 = 소수 아님)
i%j가 1,2,...이면 True (나누어지지 않음)

③ all() — 모두 True여야 소수
all(i%j for j in ...) = "모든 j에 대해 i%j ≠ 0" = "아무도 i를 나누지 못함" = 소수!

④ sum(bool, ...) 로 카운팅
sum([True, False, True]) = 2 (True=1, False=0)

a = []
for i in range(4, 9):
    a.append(i)

6

⚡ for-else 구문 (파이썬 고유 문법!)

#19 pop() + 슬라이싱 + print(*) — 역방향 추출
슬라이싱pop언패킹
▾ 
#19
celebrity = [
    '아, 얼마나 좋을까요',
    '그대란 행운이 온 걸까',
    '지금 우리 함께 있다면',
    '어떻게 나에게'
]

iu = []
while celebrity:
    iu.append(celebrity.pop())

print(*iu[1::2], sep="\n")
출력 지금 우리 함께 있다면 아, 얼마나 좋을까요
# pop()은 맨 뒤에서 꺼냄 celebrity.pop() → '어떻게 나에게' iu=['어떻게...'] celebrity.pop() → '지금 우리 함께 있다면' iu=['어떻게...','지금...'] celebrity.pop() → '그대란 행운이 온 걸까' iu=['어떻게...','지금...','그대란...'] celebrity.pop() → '아, 얼마나 좋을까요' iu=['어떻게...','지금...','그대란...','아,...'] iu[1::2] = iu[인덱스 1,3] = ['지금 우리 함께 있다면', '아, 얼마나 좋을까요']
⚡ 슬라이싱 [시작:끝:간격]
iu[1::2]: 인덱스 1부터 시작, 끝까지, 2칸씩 건너뜀

#20
upper = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"
lower = "abcdefghijklmnopqrstuvwxyz"

def encrypt(plain_text, shifts):
    result = ""
    for ch, shift in zip(plain_text, shifts):
        if ch in upper:
            idx = upper.index(ch)
            result += upper[(idx + shift) % 26]
        elif ch in lower:
            idx = lower.index(ch)
            result += lower[(idx + shift) % 26]
        else:
            result += ch
    return result

#21 복호화 함수 (decrypt) — 시저 암호 역변환

▾ 
#21  암호문: MqyiRezmuf  회전값: 1234567890
def decrypt(cipher_text, shifts):
    result = ""
    for ch, shift in zip(cipher_text, shifts):
        if ch in upper:
            idx = upper.index(ch)
            result += upper[(idx - shift) % 26]  # +가 아닌 -!
        elif ch in lower:
            idx = lower.index(ch)
            result += lower[(idx - shift) % 26]
        else:
            result += ch
    return result
출력 (MqyiRezmuf, shifts=1234567890) LoveMyself
M: upper, idx=12, (12-1)%26=11 → L q: lower, idx=16, (16-2)%26=14 → o y: lower, idx=24, (24-3)%26=21 → v i: lower, idx=8, (8-4)%26=4 → e R: upper, idx=17, (17-5)%26=12 → M e: lower, idx=4, (4-6)%26=24 → y ← (-2)%26=24 파이썬에서 음수 mod는 양수! z: lower, idx=25, (25-7)%26=18 → s m: lower, idx=12, (12-8)%26=4 → e u: lower, idx=20, (20-9)%26=11 → l f: lower, idx=5, (5-0)%26=5 → f 결과: LoveMyself
🔴 파이썬의 음수 나머지 — (-2)%26 = 24
다른 언어와 달리 파이썬에서 %는 항상 양수를 반환합니다.
-2 % 26 = 24 (← 26-2=24)
이 덕분에 알파벳 순환 처리가 자동으로 됩니다!
